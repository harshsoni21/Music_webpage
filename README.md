# Music_webpage
Link of Website https://harshsoni21.github.io/Music_webpage/

The Spotify music app clone is a web application built with HTML, CSS, and JavaScript. It enables users to browse, search, and listen to music and experience a seamless music playback interface. The app includes a user-friendly UI with a navigation bar for Home,, Search, and My Library sections. It supports music playback using HTML5 audio along with media controls and responsive design.
